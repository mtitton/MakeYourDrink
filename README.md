# Make Your Drink 🍸

Transform the ingredients you already have at home into amazing cocktails.

Make Your Drink is an iOS app built with SwiftUI that helps users discover, create and save drink recipes using ingredients they already own. The app combines ingredient management, intelligent recommendations, OCR scanning and an AI Bartender experience.

---

# Features

## My Bar

Build your personal ingredient inventory.

Users can:

- Add ingredients manually
- Add ingredients by text
- Add ingredients from image scanning
- Remove ingredients
- Organize their home bar

---

## Smart Drink Matching

The app analyzes available ingredients and suggests recipes.

Features:

- Match percentage calculation
- Available ingredients
- Missing ingredients
- Ready-to-make drinks
- Nearly-ready drinks

Example:

70% Match

You have:
✓ Gin
✓ Lime
✓ Tonic Water

Missing:
✗ Rosemary

---

## AI Bartender ✨

Create personalized drinks using natural language.

Examples:

"Create something refreshing with gin"

"I want a sweet vodka drink"

"Make a strong whisky cocktail"

The AI Bartender analyzes:

- Preferred spirit
- Flavor profile
- Alcohol strength

And generates:

- Custom drink name
- Description
- Ingredients
- Preparation steps

---

## Saved AI Drinks

Users can save drinks generated by the AI.

Features:

- Save generated recipes
- Access later
- View complete recipe
- Personal drink collection

---

## Ingredient Scanner

Photo-based ingredient detection.

Current version:

- Photo Picker
- OCR using Vision Framework
- Ingredient recognition
- User confirmation before importing

Flow:

Photo
↓
OCR
↓
Detected Ingredients
↓
User Confirmation
↓
Add To My Bar

---

## Smart Shopping List

Shows ingredients that unlock the most recipes.

Example:

Buy:
✓ Tonic Water → unlocks 8 drinks
✓ Lime → unlocks 6 drinks
✓ Ginger Beer → unlocks 4 drinks

---

## Favorites

Users can save favorite drinks.

Features:

- Favorite recipes
- Quick access

Scale recipes for multiple people.

Example:

4 People:
200 ml Gin

## Guided Preparation

Features:

- Progress tracking
